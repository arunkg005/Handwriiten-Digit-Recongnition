A simple look at how the app turns handwritten strokes into clean, structured predictions with fast feedback and region-aware analysis.

About the Project

This web app demonstrates real-time handwritten digit recognition through an interactive canvas. It helps users see how strokes are interpreted, separated, and mapped into stable predictions.

Our Model

The predictor is powered by a local MNIST-trained MLPClassifier(hidden_layer_sizes=(256, 128)). A binary rejector then filters uncertain marks and non-digit strokes before the UI commits a final answer, while the pipeline still normalizes strokes, detects separate digit regions, and scores each active region with confidence values.
